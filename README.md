# Projeto JoKenPo - Android

## Descrição do Projeto
Aplicativo de tela única que simula o jogo Pedra, Papel e Tesoura. O usuário joga contra o computador em um sistema de **melhor de 3**. 

## Como Funciona
1. O jogador clica em uma das três opções (Pedra, Papel ou Tesoura).
2. O computador escolhe uma opção aleatoriamente.
3. O app compara as jogadas e atualiza o placar (PC x YOU).
4. O primeiro a atingir **2 pontos** é declarado o vencedor no campo de resultado.
5. O botão **Reiniciar** zerá o placar e limpa o resultado para uma nova partida.
